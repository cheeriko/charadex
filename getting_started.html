Setting Up Your Sheet

Information is presetup for your using pleasure, but we still need to get it ready to use on the site.

Step One: Go to File and select Publish to Web.
Step Two: Select the Masterlist document ONLY.
Step Three: Publish!

Permission Warnings
If you ever need to add mods into the sheet, do NOT allow everyone to edit via link. That will allow anyone to come in and edit whatever they like. You need to click Share and add people via their emails instead.

Hosting your Site

I will only be showing you how to host on neocities. It is probably easier to host on github using gitpages but you need to know how to use github to really utilize it. I might make a tutorial later, but unfortunately right now I don't have the time :'3c Without further ado.

Make sure you've downloaded the .zip to charadex. Here's a quicklink if you haven't.

First, you'll need to sign up to Neocities. Then go to Edit Site, you'll see a menu with 4 files. Delete every file except the Index file.

Unzip the charadex file, then open the folder named charadex-master and highlight all the files. Drag and drop them into the Neocities dashboard.

You should now have an up and running site!!!

Importing your Sheet

Right now, the site should be importing from my example sheet. Grab your sheet ID - it's located in the URL.

Example: /spreadsheets/d/1vVtBIK-ftiPTTzPnn40uLAafWJrGvqIrW1GFYo1gmrM/edit#gid=0

Edit the Index file and scroll to the bottom of the Index file, is a code that says var sheetID = "1vVtBIK-ftiPTTzPnn40uLAafWJrGvqIrW1GFYo1gmrM" replace this ID with yours and save. You should be good to go!

If you have shifted the sheets around for whatever reason, sheetPage needs to be the page number of the sheet. Like right now, it's the 3rd sheet in the document - if you move it right behind Options, it would be the 2nd. So the number would change to 2.
