Customizing your Site

I suggest customizing your sheet before trying to customize your site. If you haven't alread, please go here for more information.

If you add more items/traits to your database, or just want to change the general site look, you'll need to know how the HTML is rendered and how List.js works.

I highly recommend you look at List.js docs and Bootstrap 5 docs while setting up and editing.

Masterlist Options

There is a small portion of script called 'Masterlist Options', in it you have a couple of choices in how to render it out. Any further edits will require JS and List.js knowledge (I'm going to sound like a broken record here, I'm sorry LOL)

Options

  • numOfItems - The amount of masterlist gallery items you want displayed on a page. I suggest no more than 32.
  • order - The order you want items to be in. From 1-2-3 to 3-2-1. Options are asc and desc.

Adding Traits/Other information

I've set up the JS to automatically add whatever you add to the sheet into the List.js settings to be rendered in the main card.

The only things rendered in the masterlist gallery are a link to the card, id number, image, owner (hidden), designer (hidden), and artist (hidden). I've added owner, designer, and artist so people can search for them.

If you need to add more items, you must call them with a class. Below is a break down of an item in a masterlist entry card.

<div class="col-md-6 d-flex justify-content-between p-2">
  <b>Owner</b> <span class="owner"></span>
</div>

All this means is that if you wanted to add another trait or another piece of info, you'd just have to insert something like this:

<div class="col-md-6 d-flex justify-content-between p-2">
  <b>Trait Name</b> <span class="traitname"></span>
</div>
